# Local full-text artifacts

PDF/XML files are stored under deterministic paper and SHA-256 directories. They are local evidence and are ignored by Git by default.

Each artifact directory contains:

- `source.pdf` or `source.xml`
- `source_manifest.json`

Do not edit source files. A lawfully obtained institutional or author copy may be imported with `rights_status=local_research_only`; this does not grant redistribution rights.
