Citation extraction from the Deep Research report: "Underutilized Computer Vision Datasets in Agriculture for Research Novelty".

Files:
1. agri_cv_dataset_citations.csv
   - Dataset/data-source citations, including official data repositories, companion repositories, and dataset-descriptor papers when the paper itself is the main dataset citation.

2. agri_cv_dataset_paper_citations.csv
   - Papers associated with the datasets. If a Scientific Data / CVF / MDPI paper is also the practical dataset citation, it appears in both the dataset and paper CSVs.

3. agri_cv_other_report_citations.csv
   - Other cited sources used in the report that are not the dataset citation or dataset paper itself, such as supporting tool repositories, repository metadata pages, or author/profile pages.

Notes:
- URLs are kept as plain text for spreadsheet import and citation checking.
- Some fields use the report metadata plus source titles visible in the report; verify final author lists/DOIs against the source pages before submitting a manuscript bibliography.
