# Curated paper-priority state

This directory stores append-only AI semantic score events and deterministic active snapshots for full-text scheduling.

Expected generated files:

- `paper_priority_score_history.csv` — append-only semantic score events;
- `paper_priority_scores.csv` — derived active score per candidate;
- `paper_priority_score_batches.csv` — batch provenance and input hashes.

Bootstrap estimates are generated outputs, not curated evidence, and are not written here.

A priority score never changes title/abstract inclusion status.
