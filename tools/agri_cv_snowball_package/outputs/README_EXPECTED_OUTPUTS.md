# Expected outputs after running the snowballing script

This folder is initially empty except for this README. After running the script, it should contain:

- `seed_papers_provider_metadata.csv`
- `seed_papers_merged_metadata.csv`
- `backward_references.csv`
- `forward_citations.csv`
- `snowball_edges.csv`
- `snowball_nodes.csv`
- `run_summary.csv`
- `unresolved_seeds.csv`
- `run_manifest.json`
- `cache/`
- `raw_seed_records/`

Use `run_summary.csv` and `unresolved_seeds.csv` first to verify quality.
